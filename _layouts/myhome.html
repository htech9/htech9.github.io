My journey

2014: Starting professional experience and research project.

Back after obtaining my bachelor degree, I joined a research lab as Java developer where I wrote a desktop application that shows image from IP cameras and record video when motion are detected. I made that with Java, my first serious programming langugage for making general purpose project.

2015: Joining a small web agency

In parallel of finishing my master degree, I joined a small web agency of a friend who is an alumni of my university. From there I participated on web projects and mobile projects development, UI integration I mainly used Php for web (CodeIgniter), Javascript (Cordova/AngularJS) or pure Java for mobile application (Android).

2016 - 2017: Started freelancing and consultance own my own

I got my master degree in december 2015. Then I decided to go freelancing on my own, working with my first clients to help them building websites and mobile application for their business. The main technologies I used are Php (CodeIgniter, Symfony), Javascript(Angular, NodeJs).

2018: Starting working at company

On that year, I got a work contract that required me to move in an european country but at the last minute I wasn't able to. And so I decided to switch on working on a local company.
On my surprise, how things works there was different from my earlier habit because process are more structured, and decisions taking are shared by more people. At the end it helped me to be a better team-player, and see some leadership model. I used mainly Javascript in that place (NodeJs, Angular).

2019 - 2022: In search for senior level and taking more responsability

I switched to another company, a social startup in healthtech sector. I was lucky to get mentored by some strong people both technically and professionally and so I felt growing 10x.
I could lean, participate, and grow in another technical area: AWS cloud infrastructure management and configuration using IaC (Terraform), containerization at deep level (docker), pipeline setup and improvement (circle ci, fastlane), bridging Javascript with native Kotlin Android application, and lot more
I could also develop soft skills: mentoring, leadership, communication, ... . All of these enabled me to work efficiently with both technical and non technical persons (...) That was enabling me through time to reach senior level and take more and more responsability by gradual promotion, and so make impact in the company. The main languages use there was Ruby (Ruby on Rails, Sinatra) , Javascript (Vue). I got also few experience with Kotlin and Java
